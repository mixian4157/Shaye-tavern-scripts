# 正则表达式配置说明

## `<StatusPlaceHolderImpl/>` 占位符的正则表达式

如果你在消息中使用 `<StatusPlaceHolderImpl/>` 作为占位符，应该使用以下正则表达式：

```regex
<StatusPlaceHolderImpl\s*/>
```

### 解释：
- `<StatusPlaceHolderImpl` - 匹配字面量开始标签
- `\s*` - 匹配零个或多个空白字符（空格、制表符等）
- `/` - 匹配自闭合标签的斜杠
- `>` - 匹配结束的尖括号

### 转义说明：
在 JSON 配置文件中，反斜杠需要转义，所以应该写成：
```json
"findRegex": "<StatusPlaceHolderImpl\\s*/>"
```

## 其他常用的正则表达式示例

### 1. 匹配包含特定关键词的消息
```regex
.*(健康状态|显示状态|状态栏).*
```
这会在消息包含"健康状态"、"显示状态"或"状态栏"时触发。

### 2. 匹配所有消息（当前配置）
```regex
.*
```
这会匹配所有消息，界面会在每条消息中显示。

### 3. 精确匹配占位符（推荐）
```regex
<StatusPlaceHolderImpl\s*/>
```
这只会匹配消息中的 `<StatusPlaceHolderImpl/>` 占位符。

### 4. 匹配多种占位符格式
```regex
<StatusPlaceHolderImpl\s*/?>|&lt;StatusPlaceHolderImpl\s*/?&gt;
```
这会匹配：
- `<StatusPlaceHolderImpl/>`
- `<StatusPlaceHolderImpl />`
- `&lt;StatusPlaceHolderImpl/&gt;` (HTML 转义形式)

## 配置文件选择

根据你的需求，选择对应的配置文件：

1. **`界面-莎夜-占位符.json`** - 匹配 `<StatusPlaceHolderImpl/>` 占位符
2. **`界面-莎夜-关键词.json`** - 匹配包含"健康状态"等关键词的消息
3. **`界面-莎夜.json`** - 匹配所有消息（当前配置）

## 使用方法

1. 在酒馆助手中导入对应的配置文件
2. 确保 `disabled: false`（已启用）
3. 在消息中使用对应的触发方式：
   - 使用占位符：在消息中输入 `<StatusPlaceHolderImpl/>`
   - 使用关键词：在消息中输入"健康状态"、"显示状态"或"状态栏"
